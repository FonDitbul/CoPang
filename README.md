# CoPang

## Description

CoPang 프로젝트 

이커머스 토이 프로젝트


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## lint

```bash
# 린트 체크
$ npm run lint

# 린트 적용 
$ npm run lint:fix
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
